# Luna Accessories

موقع عربي بسيط لبيع الاكسسوارات مع نموذج طلب يرسل البيانات إلى الإيميل.

## تعديل الإيميل

افتحي ملف `settings.js` وعدلي السطر التالي:

```js
orderEmail: "your-email@example.com",
```

وضعي مكانه الإيميل الحقيقي الذي تريدين استقبال الطلبات عليه.

## كيف يعمل الإرسال

الموقع يستخدم خدمة `FormSubmit` لإرسال بيانات الطلب من الفورم إلى الإيميل مباشرة.

1. عند أول طلب غالبًا ستصلك رسالة تفعيل من الخدمة على الإيميل.
2. بعد تفعيل الإيميل، أي طلب جديد من الموقع سيصل لك على نفس الإيميل.

## ملفات المشروع

- `index.html`: الصفحة الرئيسية للمتجر.
- `styles.css`: التصميم والألوان والاستجابة للموبايل.
- `app.js`: اختيار المنتج وتجهيز الفورم.
- `settings.js`: إعداد الإيميل الذي يستقبل الطلبات.

## تشغيل الموقع

بما أن المشروع ثابت، يمكنك فتح `index.html` مباشرة أو رفع الملفات على أي استضافة ثابتة مثل:

- GitHub Pages
- Netlify
- Vercel
